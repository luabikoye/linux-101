# Introduction to Linux

## What is Linux?
Linux is a free, open-source operating system that powers most of the world’s servers, cloud platforms, supercomputers, and Android devices.

It manages:
- Hardware resources (CPU, memory, disk)
- Running applications
- Security and user access
- Networking and system processes

Unlike Windows or macOS, Linux is community-driven, highly customizable, and built for performance and stability.

## Brief History
- Created in 1991 by Linus Torvalds
- Inspired by UNIX
- Free alternative for developers and sysadmins
- Used today in:
  - AWS, Azure, Google Cloud
  - Web servers (Apache, Nginx)
  - Databases
  - Containers (Docker, Kubernetes)

## What is a Linux Distribution?
A Linux distribution (distro) is a packaged version of Linux including:
- Kernel
- System tools
- Package manager
- Optional Desktop environment

Popular distros:
- Ubuntu – beginner-friendly
- CentOS / Rocky Linux – enterprise servers
- Debian – stable and secure
- Fedora – cutting-edge
- Kali – security & penetration testing
