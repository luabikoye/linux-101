# Docker Commands

- `docker run image` → Run image
- `docker images` → List images
- `docker ps` → Running containers
- `docker ps -a` → All containers
- `docker run --name web01 -d -p 9080:80 nginx` → Run container with host port
- `docker compose up` → Start compose
- `docker compose down` → Stop compose
- `docker system prune -a` → Clean unused
