# VIM Editor Commands

## Insert Mode
- `i` → Insert at cursor
- `o` → Insert new line below

## Save & Quit
- `:wq` → Save and quit
- `:q!` → Quit without saving

## Navigation
- `gg` → Beginning of file
- `Shift + G` → End of file
- `w` → Forward one word
- `b` → Backward one word
- `5w` → Forward 5 words
- `5b` → Backward 5 words

## Copy, Paste & Delete
- `yy` → Copy current line
- `4yy` → Copy next 4 lines
- `p` → Paste
- `dd` → Delete current line
- `5dd` → Delete next 5 lines

## Undo & Redo
- `u` → Undo
- `Ctrl + r` → Redo

## Search
- `/word` → Search for word
- `n` → Next occurrence
