# Disk & Networking

## Disk Commands
- `df -h` → Disk usage
- `du -sh /var/log` → Directory size
- `find / -size +100M` → Large files
- `fdisk -l` → List disks
- `mkfs.ext4 /dev/nvme1n1` → Format disk
- `mount -a` → Mount all

## Networking
- `ping <IP>` → Check connectivity
- `netstat -antp` → Open ports
- `nmap <IP>` → Port scan
- `dig <domain>` → DNS info
- `route -n` → Routing table
- `arp` → ARP table
