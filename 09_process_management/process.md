# Process Management

## ps
- `ps` → Current user
- `ps aux` → All processes
- `ps aux | grep nginx` → Filter

## top
- `top` → Live monitor
- `htop` → Enhanced (if installed)
- `nproc` → CPU count
- `q` → Quit
- `k` → Kill process

## kill
- `kill PID` → Graceful
- `kill -9 PID` → Force
- `killall nginx` → Kill all
